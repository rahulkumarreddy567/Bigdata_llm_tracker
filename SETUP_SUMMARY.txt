TROUBLESHOOTING
================================================================================

Port already in use:
  netstat -ano | findstr :8080
  taskkill /PID <PID> /F

  docker-compose logs

  docker-compose up -d postgres

Can't access Airflow:
  - Wait 1-2 minutes for services to fully initialize
  - Check: http://localhost:8080/health
  - View logs: docker-compose logs airflow-webserver

Elasticsearch connection error:
  - Check: http://localhost:9200/_cluster/health
  - View logs: docker-compose logs elasticsearch

================================================================================
PROJECT OVERVIEW
================================================================================

The LLM Tracker compares LLM models by value:
  - Value = Quality / Cost
  - Quality: LMSYS Arena Elo ratings
  - Cost: OpenRouter API pricing
  - Result: Value scores for each model

Daily pipeline:
  1. Fetches latest pricing & rankings
  2. Normalizes data with DBT
  3. Combines sources into value scores
  4. Indexes in Elasticsearch
  5. Displays in Kibana dashboard

All automated, scalable, reproducible!
