🐛 TROUBLESHOOTING

Problem: "Index not found"
Solution: Check Elasticsearch has data
          curl http://localhost:9200/_cat/indices?v
          Should show: llm-value-scores with 674 docs

Problem: "Pattern matches 0 indices"
Solution: Use exact pattern: llm-value-scores*
          (lowercase, with asterisk)

Problem: "No timestamp field"
Solution: Select ingestion_date from dropdown
          (it's the only date field)

Problem: Data view not visible
Solution: Clear browser cache (Ctrl+Shift+Delete)
          Reload page
