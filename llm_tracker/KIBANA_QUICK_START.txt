## 🚀 CREATE DATA VIEW - 3 SIMPLE STEPS

### Step 1: Open Kibana
```
Browser: http://localhost:5601
Click: ☰ Menu (top left)
Select: Stack Management
```

### Step 2: Go to Data Views
```
Click: Data Views (left sidebar)
Click: + Create data view
```

### Step 3: Fill the Form
```
Index pattern: llm-value-scores*
Timestamp field: ingestion_date
Name: LLM Value Scores
Click: Save data view ✅
```
