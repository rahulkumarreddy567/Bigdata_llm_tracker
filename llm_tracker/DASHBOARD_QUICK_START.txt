## 🔧 COMMON ACTIONS

### Add Filter
```
1. Click: + Filter
2. Select: Field (provider, performance_score, etc.)
3. Choose: Condition
4. Apply: Updates all visualizations
```

### Export to PDF
```
1. Menu (⋮) → Export
2. Choose: PDF
```
